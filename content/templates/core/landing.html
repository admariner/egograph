About

When you start typing something into Google have you noticed it will try to autocomplete your query? This project uses those suggestions to create a network graph from each search query. For example, try the word blueberry.

Thoughts behind this project:
  1. Everything that humans think about they eventually type into Google.
  2. Google stores this data and (I assume) uses it to make better suggestions.
  3. Therefore, a network graph of these suggestions is a map of the human psyche.
